# Phonebook API

A simple phonebook backend built with **Node.js** and **Express.js**. This API allows you to view, add, and delete contacts. It also provides an info endpoint to check the number of entries and the current server time.

## Features

- Get all contacts  
- Get a contact by ID  
- Add a new contact  
- Delete a contact  
- Info endpoint showing the total number of contacts and current date/time  
- Request logging with **morgan**  
- Handles CORS requests  
- Serves static files from the `dist` folder
